### Download rust:
`curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`
### Download the code:
`git clone https://github.com/soldierdonkey/quicksort`
### Open the directory:
`cd quicksort`
### Open the code in:
`./src/main.rs`
### Run code (run with quicksort as the active directory):
`cargo run`
